# Описание
В рамках курсовой работы по предмету "Машинное обучение" был запущен переводчик с португальского на английский на базе архитектуры Transformer с помощью инструкций на TensorFlow и адаптирован под ограниченные ресурсы бесплатной версии Goole Colab.

# Цель
Реализация переводчика на основе архитектуры Transformer.

# Задачи
1. Запустить код из руководства TensorFlow, учитывая возможные проблемы.
2. Проанализировать влияние ограниченных ресурсов на процесс обучения.
3. Провести качественную  и количественную оценку работы модели.

# Ограничения:
- Установка специализированных библиотек (таких как libcudnn8) на сторонних инфраструктурах невозможна. В руководстве эта библиотека устанавливается.
- Доступны только стандартные вычислительные ресурсы Google Colab.
- Требуется строгое следование коду руководства как эталонной реализации.
